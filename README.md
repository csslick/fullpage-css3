# fullpage-css3
## Smooth page scroller
- by CSSLICK 
- release date: 2019.12.25

## Usage
```
    <nav class="page-link">
      <a href="#page1">1</a>
      <a href="#page2">2</a>
      <a href="#page3">3</a>
    </nav>
    <div id="page">
        <section class="page" id="page1">
          <h1>page1</h1>
        </section>  
        <section class="page" id="page2">
          <h1>page2</h1>
        </section>  
        <section class="page" id="page3">
          <h1>page3</h1>
        </section>
    </div><!-- end page -->

  <script src="csslick-scroll.js"></script>
```
